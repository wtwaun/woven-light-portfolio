# Published Photos

Place your Lightroom exports (JPEGs) here, organized into folders corresponding to your site categories (e.g., `Wildlife`, `Landscape`, `Stars`).

- Use the `001-OriginalName.jpg` pattern to control the sort order.
- This folder is watched by GitHub Actions: when you push new photos here, the CI will automatically generate optimized WebPs, update the JSON database, and trigger a deployment!
